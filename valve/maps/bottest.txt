---Play Information------

Map Description:

This started out as a map for testing the bots that has
been released for Half-Life. I made a simple version of
the biggest room in the map and tried it out with Jumbot and
Rhobot. Then I added some more rooms and features....

Single Player: 

None, but you can start a simple player game and walk a
little around, shoot some barneys. These where put in 
for testing purposes.

Deathmatch: Yes

Teamplay:   Yes

Recommended # of Players:  6+
